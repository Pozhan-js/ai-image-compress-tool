# 按页面与功能模块划分的实现步骤

## 页面 A：工作台（主页面）

### A1. 上传区模块
- 支持拖拽、点击选择、多选、文件夹导入。
- 前端先行校验：格式、单图大小、批量数量。
- 生成本地预览与基础元数据（尺寸、格式、大小）。
- 输出：待处理文件列表。

### A2. 参数配置模块
- 提供三种模式：智能、预设（高质量/平衡/高压缩）、自定义。
- 自定义支持：质量、尺寸、格式转换、无损/有损。
- 参数变更触发预估提示：预计体积区间、风险提醒。
- 输出：压缩任务参数对象。

### A3. 任务执行与进度模块
- 点击开始压缩后创建任务。
- 显示总进度 + 单文件进度 + 当前处理文件名。
- 支持暂停/取消（首版可先做取消）。
- 失败项单独标记，可重试。
- 输出：任务结果集合。

### A4. 对比预览模块
- 单图对比：原图/压缩图分屏或滑条。
- 批量对比：左侧缩略图列表，右侧详情对比。
- 展示统计：原始大小、压缩后大小、压缩率、耗时。
- 输出：用户可感知质量与压缩效果。

### A5. 导出模块
- 单张下载与批量导出（zip）。
- 支持保存路径选择与命名规则。
- 导出完成提示并可打开目标目录。

## 页面 B：历史记录页

### B1. 历史列表模块
- 展示最近任务（时间、数量、总压缩率、状态）。
- 支持按日期、格式、大小筛选。

### B2. 历史详情模块
- 可查看任务内每张图的压缩前后数据。
- 支持重新导出、按原参数再次压缩。

### B3. 清理与保留策略模块
- 支持按天数自动清理历史记录。
- 支持手动批量删除历史项。

## 页面 C：设置页

### C1. 通用设置模块
- 主题切换、语言（中文优先）、默认输出目录。

### C2. 压缩默认配置模块
- 默认预设、默认输出格式、EXIF 保留策略。

### C3. 性能与隐私模块
- 并发线程数上限、临时目录位置、自动清理开关。
- 明确提示“全部本地处理，不上传服务器”。

## 实现顺序（建议）
1. 页面 A 的 A1 + A2（先打通输入与参数）。
2. 页面 A 的 A3（接入后端压缩与进度）。
3. 页面 A 的 A4 + A5（形成完整价值闭环）。
4. 页面 B（历史查询与复用）。
5. 页面 C（默认配置、性能、隐私）。
6. 统一异常处理、测试补齐、打包发布。
